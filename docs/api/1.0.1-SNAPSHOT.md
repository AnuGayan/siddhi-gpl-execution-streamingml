<p style="word-wrap: break-word">An AMRulesRegressor model with the name 'model1' will be built/updated with a split confidence of 1.0E-7D, tie break threshold of 0.05D, a grace period of 200, NoChangeDetection and NoAnomalyDetection using attribute_0, attribute_1, attribute_2, attribute_3 as features and attribute_4 as the target value. The accuracy evaluation will be emitted to the outputStream.</p>
